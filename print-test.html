Print Instructions:

  1. Press Ctrl+P (or Cmd+P on Mac) to open print dialog
  2. IMPORTANT: Look for "More settings" or "Options"
  3. Find the "Scale" setting and make sure it's set to 100% or "Default"
  4. Disable "Fit to page" or "Shrink to fit" if present
  5. Check the preview - you should see 2-3 pages

If you still see only 1 page: Your browser is forcing content to fit. Try using Firefox or Edge instead of Chrome.
